Example Gallery

The repository has 17 finished example directories under examples/, plus standalone specs under specs/. This page is the map — it does not duplicate the examples themselves. Links go to the source on GitHub.

Start here

gallery/ — "read FSL small, try it big" material

Each .fsl file writes expected-command / expected-result / expected-kind in comments, and tests/test_gallery.py matches them against the actual fslc JSON.

Flagship examples

One domain, all three layers

e2e/

The flagship example: a consultant, a PM, and an engineer write the same expense-reimbursement domain at different granularities, connected as business ⊒ requirements ⊒ design ⊒ implementation.

layers/

The final form of DESIGN-layers.md. The return domain is written in three dialects and chained by refinement.

bank/

Wires a plain Python implementation (bank.py) with no knowledge of FSL to implementation-conformance checking via replay/testgen.

refinement_chain/

Confirms propagation through a layer chain (business ⊒ requirements ⊒ design …).
