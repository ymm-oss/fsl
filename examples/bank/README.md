# bank — A worked example of implementation-conformance testing

An example that wires a plain Python implementation (`bank.py`) with no knowledge
of FSL to a harness generated by `fslc testgen` from a spec
(`specs/bank_system.fsl`, already composed and proved), using a small (~20-line)
Adapter. All 8/8 checks pass: 7 replayed scenarios plus a 100-step random walk
with the Monitor as the oracle.

```bash
cd examples/bank && PYTHONPATH=. ../../.venv/bin/python -m pytest test_bank_conformance.py -q
```
